      Keterangan Sistem

      Tentang Sistem

      Sistem informasi pengelolaan satuan pendidikan non formal SKB/PKBM tahun 2026 di lingkungan Dinas Pendidikan Kabupaten Sumedang. Versi single-file ini di-host di GitHub Pages dan menyimpan data secara lokal di browser pengguna (localStorage).

      Kategori Data

      1. Data Akreditasi – Status akreditasi seluruh lembaga PKBM/SKB beserta tahun berlaku dan nomor SK.
      2. Data Peserta Didik SPMB ATS/APS – Rekapitulasi peserta didik baru jalur ATS/APS, gender, dan status verval.
      3. Data E-Rapor 8 SNP SKB/PKBM – Capaian 8 Standar Nasional Pendidikan.

      Fitur

      • Grafik laporan otomatis di Beranda untuk 3 kategori data.
      • Tombol + Input Data dengan opsi pilihan dropdown.
      • Tombol Edit & Hapus per baris.
      • Tombol Cetak, CSV, & Unduh PDF.
      • Pencarian (search) per tabel.

      Legenda Warna

      ● Baik / Sudah / Berlaku
      ● Cukup / Sedang
      ● Kurang / Belum / Habis Berlaku
      ● Informasi / Tautan

      💾 Backup & Restore Data

      Simpan seluruh data ke file JSON sebagai cadangan, atau pulihkan dari file backup sebelumnya.

      💡 Tip: Backup secara berkala karena data tersimpan di browser & bisa hilang jika cache dihapus.
